## What Nudge Does
- **Email Logs** — AI categorizes every email automatically
- **Work Board** — Kanban with AI bottleneck detection and task prediction
- **Schedule** — AI builds your optimal day from emails and calendar
- **Inner** — Real-time team collaboration
- **Business OS** — Owner dashboard with departments and agents
- **Nudge AI** — Floating assistant that reads emails and creates meetings

## Tech Stack
- Frontend: HTML, CSS, JS on Cloudflare Pages
- Database: Supabase
- AI: Groq API — Llama 3.3 70B
